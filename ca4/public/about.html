Introduction

Word Chat is a chat which can be accessed from all over the world. You just need to access the link, insert an username and you are joining the chat!

What you can do

  • You can access the chat using this link.
  • Before start chatting you have to insert an username. Choose the username that best fits you!
  • You are in the chat! Send your messages to the world! Everyone that is joining can read you messages and your username

Other information

Online users' list

On the left side there is the list of the online users. It shows each username and when they are writing.

Message text

When your message is ready to be sent you can press the "Send" button or just press the enter key! You can write multiple line of text, if you want start a new line without sending the message press shift+enter.

You can scroll to all the old message, be carefull of updating the page because you can lose all the messages!

Joining the chat

Everytime a user joins the chat, every online user receive a notification and it remains written in the chat.

Leaving the chat

If a user leave the chat every online user can see it written in the chat.
